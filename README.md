# 私生活ログ

朝から夜までのスケジュール、食事、運動、支出、スマホ利用時間を日記形式で記録し、
自動集計・グラフ表示する私生活管理ツールです。

## スマホ・どこからでもアクセス

`main` にプッシュされると GitHub Actions が自動でビルドし、GitHub Pages に公開します。
公開URL: `https://v5jpksws79-commits.github.io/-/`

初回のみ、リポジトリの **Settings > Pages > Build and deployment > Source** を
「GitHub Actions」に設定してください(手動での一度きりの設定です)。
公開後はスマホのブラウザからこのURLを開き、「ホーム画面に追加」するとアプリのように使えます。

## 特徴

- **日記風の自由記述**: 1日の出来事を文章で書くだけ。特定のタグを混ぜて書くと、
  支出・食事・運動・スマホ利用・スケジュールとして自動的に抽出・集計されます。
- **ローカル保存のみ**: データはブラウザの localStorage に保存され、外部サーバーには送信されません。
- **集計・グラフ**: 直近7/14/30日間の支出・運動時間・スマホ利用時間などを日別グラフで確認できます。

## 記法

```
7:00 起床                          ← 時刻から始まる行は自動でスケジュールに
7:30 #食事 朝食 パンとコーヒー (約350kcal)  ← 朝食/昼食/夕食/間食/夜食 + 内容。「◯◯kcal」でカロリー自動集計
12:30 #支出 食費: 800円 ランチ定食   ← カテゴリ(省略可): 金額円 メモ
19:00 #運動 ランニング 30分         ← 種目 + 時間(分)
21:00 #スマホ YouTube 45分         ← アプリ・目的 + 時間(分)
```

アプリ内の「記法ヘルプを見る」からもいつでも確認できます。

## 音声入力での記録(Claude Code スキル)

スマホの音声入力(キーボードのマイクボタン)で自然にしゃべった内容をそのまま貼り付けても、
タグが付いていないため自動集計はされません。このリポジトリには
`.claude/skills/diary-format/` というスキルを用意しており、Claude Codeでタグなしの
日記文章を貼ると、上記の記法に沿ったテキストへ変換してくれます(食事は一般的な栄養知識から
おおよそのカロリーも見積もって `(約◯◯kcal)` を付記します)。変換結果をそのままアプリの
記録欄にコピペするだけで、スケジュール・食事・支出・運動・スマホ利用として自動抽出されます。

## 開発

```bash
npm install
npm run dev      # 開発サーバー起動
npm run build    # 本番ビルド
npx vitest run    # パーサーのユニットテスト実行
```
